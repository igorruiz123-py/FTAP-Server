Significado dos códigos de comunicação cliente-servidor.

Códigos de sucesso e status (cliente):

100 -> cliente abre socket com servidor 
200 -> cliente envia nome de usuario ao servidor 
300 -> cliente envia senha do usuário ao servidor 
400 -> cliente envia mensagem ao servidor 
500 -> cliente fecha o socket com servidor

Códigos de sucesso e status (servidor):

120 -> servidor aceita socket com cliente
220 -> servidor aceita nome de usuário do cliente
320 -> servidor aceita senha de usuário do cliente
420 -> servidor envia confirmação de recebimento de mensagem do cliente
520 -> servidor envia confirmação do fechamentp do socket com o cliente

Códigos de erros (servidor):

610 -> login recusado pelo servidor
620 -> comando não conhecido
