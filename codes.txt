Significado dos códigos de comunicação cliente-servidor.

Códigos de sucesso e status:

100 -> cliente abre socket com servidor | 120 -> servidor aceita socket com cliente
200 -> cliente envia nome de usuario ao servidor | 220 -> servidor aceita nome de usuário do cliente
300 -> cliente envia senha do usuário ao servidor | 320 -> servidor aceita senha de usuário do cliente
400 -> cliente envia mensagem ao servidor | 420 -> servidor envia confirmação de recebimento de mensagem do cliente
500 -> cliente fecha o socket com servidor | 520 -> servidor envia confirmação do fechamentp do socket com o cliente

Códigos de erros e status:

600 -> nome de usuário recusado pelo servidor
610 -> senha recusada pelo servidor
620 -> comando não conhecido
